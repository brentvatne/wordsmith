# Markdown Basics This is a cheatsheet of Github Flavored Markdown. *To get started, feel free to overwrite this with your own document, or click __new__ in the top right corner.* ## Formatting ### Paragraphs A paragraph is any "regular" text block with blank lines before and after. Paragraphs can have `inline code`, _italics_, __bold__ and ~~strike through~~. It also supports , named [hyperlinks](http://example.com) and simple hyperlinks http://example.com. ### Quotes > Markdown also has quoted text blocks > that make pretty blockquotes. ### Unordered lists * Milk * Eggs ### Ordered lists 1.
